# Opage

> Homepage for your browser

`Opage` 是一个客制化上网首页

基于 Material Design & Vue.js

集成搜索，网站，天气，TODO

Faster & Pure

## 功能   

- 搜索：百度 + Google + Bing
- 网站：可自定义网站列表
- 天气：南京 + 4 天预测
- TODO: 简单的 Todo 列表

## 部署使用

```bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# 切换到dist目录直接打开即可看到效果
cd ./dist/
```
